# User Behavior Analytics at Scale

## Executive Summary

A SQL-based analytical engine designed to calculate User Lifetime Value (LTV) and Churn Propensity across a dataset of 10M+ records. This project focuses on high-performance query optimization and complex cohort analysis.

This project analyzes customer lifecycle dynamics using a large-scale telco dataset to identify drivers of churn and calculate Long-Term Value (LTV). By moving beyond static reporting into Behavioral Cohort Analysis, this project provides actionable insights into customer retention and revenue optimization.
